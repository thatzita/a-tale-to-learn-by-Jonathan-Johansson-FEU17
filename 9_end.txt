Mr Git gick iväg lycklig till Hackermans skrivbord och började ladda upp alla hans filer på github. 
Hans var nöjd med hans jobb. Mr Git var inte riktigt nöjd med sin lön men det var Hackerman. Hackerman var också nöjd över att äntligen ha en som kan ladda upp hans filer. 
Alla arbetade lyckligt vidare på Zoocom tills Mr Git gick in i väggen.
